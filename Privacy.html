Privacy Policy for ChatMark

Effective Date: August 26th, 2025

Introduction

ChatMark respects your privacy. This extension is designed to help users save and organize snippets of text from ChatGPT conversations. We do not collect, transmit, or sell any personal information.

Data Collection and Use

Personal Data: ChatMark does not collect or process any personally identifiable information (such as names, email addresses, or payment details).

Website Content: The extension only processes text within ChatGPT conversations that the user explicitly selects.

Storage: All saved snippets are stored locally in your browser using the Web Storage API (localStorage). This data remains entirely under your control and is not transmitted to external servers.

Data Sharing

ChatMark does not share, sell, or transmit any data to the developer, third parties, or external services. No data leaves your device.

Permissions

ChatMark requires permission to run on https://chatgpt.com/c/* so it can function on ChatGPT conversation pages. This access is strictly limited to enabling the bookmarking feature and does not involve collecting any unrelated information.

Remote Code

ChatMark does not use or load any remote code. All code is packaged with the extension in compliance with Chrome Web Store policies.
